hexo+github 搭建博客系统

发表于 2019-02-21

准备工作

  • 一个有效的Github账号
  • 本地有配置node.js 、 npm
  • 安装了git for windows

博客搭建

1.创建仓库

在GitHub 上新建一个仓库，名称为你的用户名.github.io

2. 配置 SSH key

直接使用用户名+密码的方式提交代码不太安全，所以我们使用 ssh key 来解决本地和服务器的链接问题

  1. 运行 git bash ，检查本机已经在的ssh密钥

如果提示，No suvh file or directory 说明是第一次使用git

  1. 生成密钥文件

连续三次回车，最终会在用户目录C:\Users\SQ_BXZ.ssh 生成密钥文件，

  1. 记事本打开 id_rsa.pub,并复制里面的内容
  2. 打开github 主页，进入 Settings -> SSH and GPG keys -> New SSH Key
    将刚复制的内容粘贴到key中，title随便填，然后保存
  3. 测试是否成功

运行上述命令，成功之后会显示
